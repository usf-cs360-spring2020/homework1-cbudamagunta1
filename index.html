Air Traffic Passenger Statistics dataset

There are 12 different columns in this dataset. Each visualization used a different perspective of the original dataset. The ones used for these visualizations include:

  1. Passenger Count: Number of passengers,
  2. GEO Region: Plain text regions.
  3. Activity Period: Number deliminating date.
  4. Price Category Code Plain text indication of low fare or other.
  5. GEO Summary Plain text indication of domestic or international travel.
